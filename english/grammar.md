# NoteBooks-Grammar
----
## 错题
1. Melbourne *has topped*(top) the list for the fifth year running.
2. Yet they're hard *to beat*(beat). 
    - ***be + 表语 表被动的时候不定式用主动*** 
3. They seemed *to be looking*(look) for trouble all the time.
    - seem
        - seem to do 要做
        - seem doing 一直，正在
4. 宾语后移
    - e.g make sth. possible -> make possible sth.
5. 表条件:
    - unless/if -> 主将从现...
    - *when*
